1.选择Spring Initializr，点击next

2.输入Group和Artifact后选择 next，

3.按照需求勾选相应的，这里我选择一个web项目

4.点击next，然后点击finish

5.项目结构如图所示，然后我们编写一个controller来测试

这里要注意：

Application启动类的位置.要将Application类放在最外侧,即包含所有子包
原因:spring-boot会自动加载启动类所在包下及其子包下的所有组件.

否则会报错：This application has no explicit mapping for /error, so you are seeing this
